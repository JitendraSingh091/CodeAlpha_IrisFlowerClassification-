# 🌸 Iris Flower Classification using Machine Learning

## 🧠 Overview

This project focuses on building a machine learning model that can classify iris flowers into three different species: **Setosa**, **Versicolor**, and **Virginica**.  

The main objective of this project is to understand how classification algorithms work and how machine learning can predict flower species based on their physical characteristics.

The model makes predictions by analyzing four important features of the flower:

- Sepal Length  
- Sepal Width  
- Petal Length  
- Petal Width  

Using Python and machine learning libraries, the dataset was analyzed, models were trained, and a simple interactive prediction app was created using Streamlit.

---

## 🎯 Project Objectives

The main goals of this project were:

- Explore and understand the Iris dataset  
- Perform data cleaning and preprocessing  
- Visualize relationships between different flower features  
- Train machine learning classification models

## 🤖 Machine Learning Models Used

Different classification models were trained and tested:

- Logistic Regression  
- Decision Tree Classifier  
- Random Forest Classifier  
- Support Vector Machine (SVM)  
- K-Nearest Neighbors (KNN)  

Each model was compared to identify the best-performing algorithm.

---

## 📈 Model Performance

Models were evaluated using:

- Accuracy Score  
- Precision Score  
- Recall Score  
- Confusion Matrix  

### Best Results

The best-performing models were:

- Random Forest Classifier  
- Support Vector Machine (SVM)  

These models achieved accuracy close to **97% to 100%**, showing excellent classification performance.
